# CodeLingo

Claude Code slash commands for understanding unfamiliar code, available in four languages.

**Read the README in your language:**
[한국어](README.ko.md) · [English](README.en.md) · [日本語](README.ja.md) · [中文](README.zh-CN.md)

## Overview

- 한국어: 낯선 코드를 빠르고 안전하게 이해하기 위한 Claude Code 슬래시 커맨드 모음입니다.
- English: Claude Code slash commands for understanding unfamiliar code, quickly and safely.
- 日本語: 見慣れないコードを素早く安全に理解するための Claude Code スラッシュコマンド集です。
- 中文: 用于快速、安全地理解陌生代码的 Claude Code 斜杠命令集。

## Commands

`/explain-file` · `/change-impact` · `/handoff`

## Install

```bash
npm install -g codelingo
codelingo install
```

Open one of the language-specific README files above for full documentation.
